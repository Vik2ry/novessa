## Deployment Flow

```
Your Code Repository (GitHub)

    Render Detects Push
         ↓
  Reads render.yaml
         ↓
  Creates Build Container
  (pip install, collectstatic, migrate)
         ↓
  Runs Deployment
  (gunicorn starts, connected to PostgreSQL)
         ↓
  Health Check Passes
  (/api/v1/health/ returns 200)
         ↓
  🚀 Live on Render
```
